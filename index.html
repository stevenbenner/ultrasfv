What is UltraSFV?

UltraSFV is a free, easy to use file verification / error checker software application for Microsoft Windows and may be the friendliest file validation application there is. It is implemented as a stand alone Windows program that will let you check file integrity using MD5 hashes or the more common CRC-32 checksums that are often included as .sfv files or embedded in file names.

CRC stands for Cyclic Redundancy Check, and is a method used to check files for corruption that may occur when transferred over unreliable channels, such as over the internet/LAN or when burning to CD/DVD media. SFV, or Simple File Verification, is the implementation of 32-bit CRCs for whole files.

This method of error detection is particularly good at detecting common errors caused by noise in transmission channels and has become very popular for checking files for such corruption and errors.

UltraSFV also supports MD5 checksums. MD5 is an even more accurate mechanism for verifying files. MD5 doesn't get used as often as CRCs but it is a powerful algorithm that can find more than just corruption. MD5 is usually used when you need to be sure that a file hasn't been modified by a hacker or virus.
