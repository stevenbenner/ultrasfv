Help and Support

Since this is a free, one-man project, there are no 24-hour help lines for you to call. However, there is documentation for general usage instructions and online message forums for you to ask questions, post bug reports and to suggest new features and improvements.

UltraSFV User Manual

Browse the UltraSFV User Manual for an introduction to the software, quick start guide, tutorial and useful hints and tips. If you would like to see more information added to the documentation resources please say so on the message forums.

Message Forums

The Message Forums are provided so you can post comments, report bugs, request new features and functionality or just chat about related (or unrelated) topics. This is also the only way to reach the developer for questions and answers.

Frequently Asked Questions

A list of common questions and answers can be found in the FAQ. Please note that the FAQ contains answers, but is not the place to ask questions. For that you need to go to the message forums. When there have a good answer to a good question it will get posted in the FAQ.

Bug Reporting

For information about bug reporting please read the Report a Bug page. This page will give you some basic information about what a bug report should contain and tips for writing helpful reports. After you have read this page proceed to the message forums and report the bug to the developer.
